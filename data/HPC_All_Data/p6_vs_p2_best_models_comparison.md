## Interpretation

This comparison shows the effect of **P** (number of subjects left out) on model performance variance:

1. **P=2 (Leave-2-out)**: Higher variance (wider IQR) - fewer subjects in training set makes performance more dependent on which specific subjects are left out
2. **P=6 (Leave-6-out)**: Lower variance (narrower IQR) - more subjects in training set provides more stability
3. **Both ANOVA and PCA** show the same P-driven variance effect, indicating this is a fundamental property of the leave-P-subjects-out cross-validation strategy
