# Classification Success Rate Variance Analysis

## Methods

### Calculation Method

For each combination of test groups:

1. For each subject in the selected groups, determine if accuracy > 50% (binary: 1 if yes, 0 if no)

2. Calculate mean: `avg(group1.subjects.above50percent + group2.subjects.above50percent + ...)`

3. This gives the **proportion of subjects correctly classified** (>50% accuracy)

4. Calculate variance of these proportions across all combinations


**Key Difference**: This analyzes variance in **classification success rate**, not variance in accuracy values.


### Sampling Strategy

- **30 random combinations** sampled per number of groups (instead of all possible combinations)

- **Classification Threshold**: 50% accuracy

- **Maximum Test Groups**: 15


### Data Source

- Individual subject accuracies from `test_predictions.parquet` files

- Per-subject accuracy calculated as: `(label == prediction).sum() / total_epochs`
