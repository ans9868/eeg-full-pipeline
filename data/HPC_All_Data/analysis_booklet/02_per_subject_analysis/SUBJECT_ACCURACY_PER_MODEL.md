# Subject Accuracy per Model

## Overview

This page documents the **subject accuracy per model** CSV, which gives, for each experiment, each subject, and each model, the median and mean accuracy (across folds) and the number of folds. It is derived from the same per-subject accuracy computed from `test_predictions.parquet` (see [ANALYSIS_METHODOLOGY.md](ANALYSIS_METHODOLOGY.md)).

## Data File

- **File:** `per_subject_classification_analysis/subject_accuracy_per_model.csv`
- **Generated by:** `build_subject_accuracy_per_model_csv.py` (run after `calculate_per_subject_accuracy_from_parquet.py`)

## CSV Schema

| Column            | Description |
|-------------------|-------------|
| `experiment`      | Experiment name (e.g. ANOVA_L_6_Uniform, PCA_L_2_Random) |
| `subject_id`      | Subject ID (sub-1 … sub-65) |
| `model`           | Model name (KNN, MLP_(Neural_Network), SVM, XGBoost) |
| `median_accuracy` | Median of per-fold accuracies for this subject×model |
| `mean_accuracy`   | Mean of per-fold accuracies for this subject×model |
| `n_folds`         | Number of folds used (same subject can appear in multiple test folds) |

One row per (experiment, subject, model). Accuracies are aggregated across folds only; each fold×model accuracy comes from the corresponding `*_per_subject_detailed.csv` (subject accuracy in that fold for that model).

## Example Use

- Compare how each model performs for a given subject (e.g. sub-10 in ANOVA_L_6_Uniform).
- Compare a given model across subjects within an experiment.
- Build tables or plots of subject×model accuracy for the booklet or papers.

## Related Files

- **Source detail:** `per_subject_classification_analysis/{EXPERIMENT}_per_subject_detailed.csv` (Subject, Fold, Model, Accuracy).
- **Per-subject (all models combined):** `per_subject_classification_analysis/{EXPERIMENT}_per_subject_summary.csv`.
