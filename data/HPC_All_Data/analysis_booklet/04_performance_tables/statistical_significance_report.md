## Interpretation Notes

- **n.s.** = not significant at α = 0.05 (two-sided)
- **✅** = p < 0.05
- **Bootstrap CIs** are computed on the best-per-fold vector (n = number of folds).
  The all-runs median (reported in Tables B–D) pools all 3 HP configs × all folds.
  The two medians may differ slightly; the best-per-fold vector is more conservative
  and avoids inflating sample size via HP pseudo-replication.
- **Within-subject experiments** (ANOVA_W_C, ANOVA_W_F, PCA_W_C, PCA_W_F) have
  only a single split (n=1 fold per model). No meaningful statistical test is possible;
  results should be treated as point estimates, not distributions.
