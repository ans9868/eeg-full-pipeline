# Per-Subject Accuracy Analysis Summary

## Overview

This analysis extracts per-subject accuracy across all LPSO folds where each subject appears as test data. For each subject, we calculate:
- **Mean accuracy** across all folds where subject was tested
- **Min/Max accuracy** (range)
- **Swing** = Max - Min (biggest variance indicator)
- **Standard deviation** across folds

## Observations

1. **Random folds show much higher variance than uniform folds:**
   - Random: Mean swings of 6.9-15.2%
   - Uniform: Mean swings of 0.48-0.55%

2. **P=2 shows extreme swings:**
   - sub-3 (PCA P=2): 47.9% swing (28.3% to 76.2%)
   - sub-18 (ANOVA P=2): 44.4% swing (46.7% to 91.1%)

4. **ANOVA generally has higher accuracy but also higher variance:**

## Methodology

For each experiment:
1. Identify best model by median performance
2. Extract all fold directories (named like `sub-2_sub-6_sub-14`)
3. For each fold, extract test accuracy from `results.json`
4. Map each subject ID to all accuracies from folds where it appears as test
5. Calculate statistics (mean, min, max, swing, std dev)
